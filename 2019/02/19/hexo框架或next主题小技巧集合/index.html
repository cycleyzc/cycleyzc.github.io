NexT 主题

首页不显示全文, 显示部分

第一种, 修改配置

  1. 进入 themes/next 文件夹
  2. 打开_config.yml
  3. 搜索auto_excerpt, 修改auto_excerpt相关属性

第二种, 修改 post 文章写法

格式:

图标 icon

很多地方用到了 fontawesome 的 icon, 但是有时在 fontawesome 网站找到名称, 但是图标还是不显示?

拿菜单举例

按_config.yml 里面写的, 在 || 后面填 fontawesome 的 icon 名称就行了, 其实这样的说法并不准确, 看代码

/themes/next/layout/\_marco/menu/menu-item.swig

准确来说, 是对 || 进行分割, 把分割出来的第二个元素拼接到 class 的 fa- 的后面

有时候直接在 fontawesome 上查 icon 的名称, 然后填在 || 后面并不起效, 因为在 next 主题下的 css 并不是这样的名称, 直接去 css 里面查完整的类名是什么就可以了

fontawesome 的 css 目录:

/themes/next/source/lib/font-awesome/css
