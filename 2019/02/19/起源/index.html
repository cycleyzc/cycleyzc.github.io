前言

旧的博客因为一些粗心大意的原因, 使之前的文章都丢失了, 所以做了这个 github 博客, 就算是重新出发, 拿最近喜欢的诗句作为开始吧

不正常人类症候群

我不预料未来

因为未来不可预料

也不怀念过去

因为过去未曾相识

我只迷恋现在

哪怕你我只是萍水相逢

就像黑夜里转瞬即逝的闪耀

也像寂静山谷中

耳边的鸟语虫鸣

我迷恋的现在

是你浅浅的微笑

是相遇的下一句

道别的上一秒

是世界熄灭前

你给我的

最后一次心跳

by  不正常人类症候群
