void的用法

参考链接: what does void 0 mean

从一些框架的源码上会经常看到void 0的这种写法

  1. void 0 是什么意思?
    void 0 的值固定是 undefined

  2. 既然void 0 的值固定是 undefined, 为什么不直接用 undefined?
    undefined 不是js的关键字, 是可以被重写的

而 void 作为js的关键字, 是readonly的, 所以更能准确的表达 undefined 的意思

  1. 为什么是void 0?
    只是为了书写方便而已
