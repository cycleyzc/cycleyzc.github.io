提高效率或幸福度的工具

发表于 2019-03-27 | 更新于 2019-05-21 | 分类于 工具 | 评论数：

科学上网

expressvpn(多平台)

用过 lantern 和 ssr 多个厂商的服务器之后, 暂时觉得还是 expressvpn 最好用

官网链接: expressvpn


🌕🌔🌓🌒🌑🌘🌗🌖🌕🌔🌓🌒🌑🌘🌗🌖🌕

写作/笔记/阅读器

typora(mac)

一个极简的 markdown 文本编辑器

官网链接: typora

marginNote(mac)

一款支持 pdf 格式的阅读器, 也可以记笔记

官网链接: marginNote

kami(chrome 插件)

正因为是 chrome 插件, 所以可以跨平台, 很方便, 功能也很多, 而且支持 google drive 存储

商店地址: kami


🌕🌔🌓🌒🌑🌘🌗🌖🌕🌔🌓🌒🌑🌘🌗🌖🌕

云存储

有道云笔记(多平台)

基于云的笔记, 支持 markdown 写作, 免费的空间也够用

官网链接: 有道云笔记

google drive(多平台)

谷歌全家桶中的云存储, 缺点就是想用就要科学上网才能用, 还有送的 15G 空间不是太大

官网链接: google drive


🌕🌔🌓🌒🌑🌘🌗🌖🌕🌔🌓🌒🌑🌘🌗🌖🌕

日程管理/任务管理

trello(多平台)

操作简单, 直观, 还支持团队协作的看板

缺点是如果是用 google 账号来登录的, 需要翻墙

官网链接: trello

MeisterTask(chrome 插件)

功能同 trello, 但因为是 chrome 插件, 更方便些

商店地址: MeisterTask

teambition(多平台)

不仅有看板功能, 还有很多适合团队使用的功能

官网链接: teambition


🌕🌔🌓🌒🌑🌘🌗🌖🌕🌔🌓🌒🌑🌘🌗🌖🌕

代码相关

vscode(多平台)

这不必多说了, 个人偏好的代码编辑器, 之前是用 atom 的, 但 atom 确实太重了, 像 vscode 这样既轻量, 生态又强大的编辑器很好

官网链接: vscode

SourceTree(多平台)

一款可视化的代码管理工具

官网链接: SourceTree

Postman(多平台, chrome 插件)

一款强大的 api 工具, 支持多种格式的 api 调用, 也支持项目 api 的管理

官网链接: Postman


🌕🌔🌓🌒🌑🌘🌗🌖🌕🌔🌓🌒🌑🌘🌗🌖🌕

其他

BetterTouchTool(mac)

一款可以自定义触摸板手势与快捷键的工具, 可以在不同的软件定义不同的手势, 也可以在不同的软件定义不同的快捷键

官网链接: BetterTouchTool

TeamViewer(多平台)

一款远程桌面访问连接工具

官网链接: TeamViewer
